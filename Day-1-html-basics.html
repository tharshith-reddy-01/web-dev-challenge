Web Development Challenge - Day 1

Today, I learned the basics of HTML. Here are some common HTML tags and how they work:

Headings: Headings are used for titles and subtitles. They range from the largest to the smallest.

This is a big heading

This is a medium heading

This is a smaller heading

Paragraphs: The paragraph tag is used for blocks of text.

This is an example of a paragraph. It helps separate blocks of text.

Line Break: This tag adds a line break to move text to the next line.

First line of text.
Second line of text after a line break.

Center: This tag centers text or other content on the page. (Note: This tag is outdated but still used in some old websites.)

This text is centered.

Div: The div tag groups content together. You can use it to style sections of your page.

This is a div element with a light blue background.

Span: The span tag is used to style parts of text within a paragraph.

This text has a red word in it using the span tag.

Anchor: The anchor tag creates a link to another page or website.

Click here to visit an example website.

Image: The image tag displays a picture. You need to specify the source and alternative text.

Button: The button tag creates a clickable button.

Horizontal Rule: This tag adds a horizontal line to separate sections of content.

Unordered List: This tag creates a list with bullet points.

  • First item
  • Second item
  • Third item

Ordered List: This tag creates a list with numbers.

  1. First item
  2. Second item
  3. Third item
